Accent Conversion Using Phonetic Posteriorgrams

Shaojin Ding, Guanlong Zhao, Christopher Liberatore, Ricardo Gutierrez-Osuna

Department of Computer Science and Engineering, Texas A&M University, USA

Voice Conversion Audio Samples

Dataset: CMU ARCTIC dataset [1]

Systems:
  • SSR (Proposed): The sparse representation-based method that constructs the structured dictionaries using phoneme labels in training and jointly optimized the standard cost function along with the Phoneme-Selective Object Function in sparse coding at runtime.
  • GMM [2]: GMM based VC method. It used a GMM to model the joint distribution of the source speech frames and the target speech frames.
  • PSOF [3]: The sparse representation-based method that constructs the structured dictionaries using phoneme labels in training and jointly optimized the standard cost function along with the Phoneme-Selective Object Function in sparse coding at runtime.
  • HDDL [4]: The sparse representation-based method that learned the structured dictionary directly from phonetic structures in the joint source-target space with no supervision in training and selected the most likely sub-dictionary for sparse coding at runtime.
